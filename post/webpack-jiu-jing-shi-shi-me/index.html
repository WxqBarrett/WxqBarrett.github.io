一、Webpack究竟是什么？

2019-10-18

Webpack是一个模块打包工具。

例如：

// 这就是引入的一个模块。 ES Moudule的引入方式。
import XXX from 'XXX';
// CommonJS的引入方式。
var XXX = require('xxx');

我们引入模块有ES Moudule、CommonJS、CMD、AMD这几种规范。

webpack就是帮我们把引入的模块打包到一起生成一个最终的js文件。
webpack不仅仅可以打包Js模块，它还可以打包css、png、svg这些文件。所以webpack就是一个模块打包工具。
